# 隐私政策

> 最后更新日期：2026年4月23日

《AI名人传》（以下简称"我们"或"本书"）非常重视您的个人信息保护。本隐私政策旨在向您说明我们如何收集、使用、存储和保护您的个人信息，以及您享有的相关权利。

请您在使用我们的服务前，仔细阅读并了解本隐私政策。如果您有任何疑问，请通过文末的联系方式与我们联系。

---

## 一、信息收集范围

### 1.1 您主动提供的信息

当您购买本书、参与活动、联系客服或订阅资讯时，您可能会主动提供以下信息：

| 信息类型 | 用途 |
|---------|------|
| 昵称/用户名 | 账户识别、个性化服务 |
| 电子邮箱 | 订单确认、资讯推送、客服联系 |
| 微信号/手机号（可选） | 电子书发放、专属客服 |

### 1.2 自动收集的信息

当您访问我们的网站时，我们可能会自动收集以下信息：

| 信息类型 | 用途 |
|---------|------|
| IP地址 | 安全防护、地区定向内容 |
| 浏览器类型/版本 | 优化网站兼容性 |
| 访问时间/页面浏览 | 网站流量分析 |
| Cookies | 登录状态保持、偏好设置 |

### 1.3 Cookies使用说明

我们使用Cookies来：
- 记住您的登录状态
- 保存您的阅读偏好设置
- 分析网站流量和用户行为

您可以在浏览器设置中禁用Cookies，但这可能影响部分网站功能。

## 二、信息使用目的

我们收集您的个人信息仅用于以下目的：

1. **订单处理**：完成购买、发货、售后服务
2. **资讯推送**：发送本书更新、活动通知（需您主动订阅）
3. **客服响应**：解答疑问、处理投诉
4. **防盗版保护**：在数字版本中添加数字水印（详见下文）
5. **合法合规**：遵守法律法规要求

## 三、信息共享与披露

除以下情况外，我们不会与第三方共享您的个人信息：

| 情形 | 共享信息范围 |
|------|------------|
| 第三方支付（微信/支付宝） | 仅用于支付处理，不存储银行卡信息 |
| 物流配送（顺丰/中通） | 仅用于实物配送，收件人信息最小化 |
| 法律法规要求 | 应政府部门合法要求 |

## 四、数字水印与防盗版

为保护知识产权，我们在数字版本的PDF中嵌入数字水印信息。

### 4.1 水印嵌入内容

当您购买电子书时，我们将根据您的账户信息生成唯一水印标识，格式如下：

| 信息类型 | 来源 | 水印格式示例 |
|----------|------|---------------|
| 账号 | 您注册的用户名 | Ajack |
| 邮箱 | 您注册的邮箱 | Bzhang@example.com |
| 手机号 | 您绑定的手机号 | 13C8573791nn（nn代数字） |

**水印示例：`Ajack_Bzhang@example.com_13C8573791nn`**

这些信息将嵌入PDF的Metadata中，用于追踪盗版来源。

### 4.2 水印用途

这些信息仅用于以下目的：
- **追踪盗版来源**：当数字版本被非法传播时，可通过水印定位泄露源头
- **法律维权**：为维权诉讼提供证据支持
- **保护正版用户权益**：确保正版用户的合法权益

### 4.3 知情同意

您购买电子书即表示同意我们将您的账户信息用于水印追踪目的。如不同意，请勿购买数字版本。

### 4.4 数据安全

水印数据库访问权限严格限制，仅用于处理盗版投诉时由授权人员查询。

如您对此有任何疑问，请联系我们的客服。
