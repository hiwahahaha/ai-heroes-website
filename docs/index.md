# AI名人传

## 序言：AI重塑一切——走向有益的未来

人工智能，正以前所未有的速度深刻改变国家竞争格局、社会组织形态，以及每个人的学习、工作、生活等。

但这场革命不只是威胁——**它更是人类文明史上最大的机遇**。从诊断疾病到机器人进入家庭，从个性化教育到科研突破，AI正在帮助人类解决那些曾经被认为不可能解决的问题。

## 创作初衷

为什么写《AI名人传》？

缘起，周末友人相聚，谈论“龙虾”热，其间，一小朋友问，何为AI幻觉？

随后谈到当前AI的飞速发展，以及AI作为新事务，所有人，甚至国家、学校、企业都面临一个新的事务。需要从

新认识和学习。友人谈及AI书籍，大家感觉：“市面AI相关的书，要么偏向某类技术教程（枯燥难

懂），要么是商业报道（缺乏深度），又或者是学术论文（普通人看不进去）。”  而AI本身包含多个学科的知识，

近2年发展迭代明显加快。因我近年一直关注AI前沿发展资讯，读了点相关的书籍，相比略懂一点点，被推举写点资

料，圈内分享。 最后的要求分成两个：一个是适合小孩的AI启蒙可科普读物，另一个AI爱好者能加强AI知识拓展和

应用。真正理解一个领域，要从理解这个领域的人开始。因此，就有了这个系列电子的出现。当然创作过程中，借助

AI Agent（openclaw和hermes-agent）辅助创作，我负责总体构思筹划、审核、校对，大模型主要是deepseek、

doubao、minimax等。

> *"如果你想了解深度学习，就去读Hinton的故事。如果你想要理解AI的未来，就去理解那些创造它的人。"*

《AI名人传》——以人物为线索，以故事为载体，讲述AI从图灵到GPT的史诗历程，将人物生平，也系统梳理技术演进

迭代发展。
